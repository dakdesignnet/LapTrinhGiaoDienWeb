Các Kiến Thức Từ Khóa Cần Biết Trong Khi Học Lập Trình Web

  • Cùng cấp ./
  • Truy xuất ra ngoài 1 cấp ../
  • Thuộc tính attribute
  • CSS chạy từ trên xuống dưới

    Có 2 đoạn code cùng thực hiện 1 chức năng, nó sẽ ưu tiên đoạn code ở dưới.

  • div là thẻ block

    div có độ rộng 100% phần tử chứa nó, lưu ý chưa nói tới vấn đề khi sử dụng với CSS.

  • img là thẻ inline

    Thẻ này tự đóng và dùng để hiển thị hình ảnh với 2 thuộc tính src và alt.

  • alt trong SEO

    Alt là viết tắt của alternate text, dùng trong SEO và sẽ hiển thị khi hình ảnh bị lỗi hoặc sai đường dẫn.

  • span là thẻ inline

    Thường dùng cho những đoạn chữ ngắn.

  • class là thuộc tính dùng để sử dụng các class cho thẻ

    Class được dùng trong CSS để định dạng các phần tử.

  • Thẻ tiêu đề:

    h1, h2, h3, h4, h5, h6

  • H1

    Mỗi trang chỉ có tối đa 1 thẻ h1, dùng cho tiêu đề lớn của trang.

  • Fonts chữ
    • Sẽ có sẵn ở Google Fonts
    • Không có ở Google Fonts mà được mua và tải về máy
  • font-weight

    Độ đậm nhạt của chữ: 100 -> 900, normal, bold, bolder, light, regular...

  • CSS Selectors

    Tag, class, id, attribute

  • CSS Reset

    Dùng để reset CSS mặc định của các trình duyệt, và phải có đầu tiên.

  • border-box và content-box

    Content-box: Độ rộng bao gồm width, padding, và border. Border-box: Bao gồm padding và border.

  • flexbox

    Áp dụng thuộc tính display: flex vào phần tử mình muốn dàn layout.

  • Media Queries

    Sử dụng để thay đổi kiểu dáng theo kích thước màn hình, giúp tạo trang web responsive.

  • responsive

    Breakpoints: 320px, 480px, 768px, 1024px, 1200px, 1440px, 1600px, 1920px.

  • position: relative và absolute

    Các thuộc tính này dùng để xác định vị trí của phần tử trong trang web.

  • animation

    Thuộc tính dùng để tạo hiệu ứng chuyển động mượt mà cho phần tử.
